# myweb

將建置的網站放置GCP
需要額外三樣文件:
1. app.yml
2. main.py
3. requirements.txt (注意文件裡面不得有pip或pipenv等套件)

GCP建立APP Engine
1. 下載Cloud SDK
2. 初始化 gcloud init
3. 部署 gcloud app deploy
